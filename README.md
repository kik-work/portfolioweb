My Personal Portfolio

![Portfolio Banner](./banner.png)

A modern, responsive personal portfolio showcasing my projects, skills, and experience. Built with **React**, **TypeScript**, **Tailwind CSS**, **Shadcn UI**, and **MongoDB**.

---

## 🚀 About

This portfolio is designed to highlight my development work with a clean, modern UI/UX. It includes sections for:

- Home
- Experience
- Projects
- Skills
- Education
- About & Contact

**Key Features:**

- Fully responsive across all devices
- Light and dark theme support
- Smooth animations with Framer Motion
- Interactive project and skill sections

---

## 🛠 Technologies Used

- **Frontend:** React, TypeScript, Tailwind CSS, Shadcn UI  
- **Backend/Database:** MongoDB (Atlas)  
- **Animations:** Framer Motion  
- **Icons:** Lucide Icons  
- **Build Tools:** Vite (or Next.js)

portfolio/

## ⚡ Features in Action

- **Home Page:** Introduction, hero section, and latest projects.  
- **Experience:** Interactive timeline showcasing work experience.  
- **Projects:** Dynamic project cards with modals for detailed view.  
- **Skills:** Animated skill charts and progress bars.  
- **Education:** Timeline of academic achievements.  
- **About:** Personal bio and contact information.  
- **Dark/Light Theme:** Toggle between dark and light modes.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile.
